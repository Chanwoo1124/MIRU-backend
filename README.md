## 📌 サービス概要

本サービスは、日本就職を目指す韓国人学生に向けて、自己分析の重要性を理解し、質問ベースで思考を深めながら、自分の経験・価値観・志望理由を自分の言葉で説明できる状態をつくるための自己分析支援サービスです。

単に模範解答を見せたり、きれいな文章を自動生成したりすることが目的ではありません。面接で深掘りされたときにも崩れないよう、自分自身の経験を、自分の頭で整理し、自分の言葉で語れる状態をつくることを目的としています。

### 🎌 このサービスを作った背景

同じ学科の友人たちが日本就職を準備する中で、自己分析の段階で共通して大きな壁にぶつかっている姿を間近で見てきました。

日本就職に必要な自己分析の方法や情報を探す過程は、韓国人学生にとって手探りで非常に遠回りになりがちです。そんな友人たちの無駄な試行錯誤を少しでも減らしたいという思いから、このサービスは生まれました。

自己分析は単なる文章作りではなく、就職の成否を左右する核心的なプロセスです。その過程をより効率的に、そして「自分の言葉」で進められるよう、本サービスを開発いたしました。

## ⚡ 主な機能

| 機能 | 説明 |
|------|------|
| ソーシャルログイン | Google (OAuth 2.0) による簡単・安全なログイン |
| 自己分析 作成/編集 | エディタを活用した強みや経験の体系的な整理 |
| コミュニティ掲示板 | 投稿、いいね、コメント/返信によるユーザー間交流 |
| リアルタイム通知 | バッジと通知一覧による重要情報の即時確認 |
| 1:1 お問い合わせ | 管理者への直接的な問い合わせと回答確認 |
| マイページ | プロフィール、活動履歴の管理および退会機能 |
| 管理者機能 | ユーザー管理およびお問い合わせへの対応 |
